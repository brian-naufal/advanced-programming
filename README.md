# Advanced Programming

This repository contains coursework, exercises, and projects from the Advanced Programming course.

The focus of this course is on writing structured, maintainable, and scalable programs using modern programming principles.

---

# Overview

This course builds on fundamental programming knowledge and introduces more advanced concepts such as object-oriented design, abstraction, and modular programming.

---

# Topics Covered

* Object-Oriented Programming (OOP)
* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Exception Handling
* File Input/Output
* Collections / Data Structures
* Basic Design Principles
